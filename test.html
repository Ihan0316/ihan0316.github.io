핵심 목표 (MVP)

"신조어(손주어)와 어르신 말씀(할머니어)을 상호 '정서 번역'해주는 텍스트 기반 웹페이지를 완성한다."

이 애플리케이션은 1일 해커톤의 핵심 목표(MVP)를 달성하기 위한 구체적인 계획을 안내합니다. 각 섹션을 클릭하며 계획의 세부 내용을 탐색해 보세요.

핵심 기능

프로젝트의 성공을 위해 반드시 구현해야 하는 핵심 기능입니다.

  • ✔ 텍스트 입력/출력: 사용자가 원문을 입력하고 번역된 결과를 볼 수 있는 UI
  • ✔ 통역 모드 선택: '손주어→할머니어' (Mode A), '할머니어→손주어' (Mode B) 2가지 모드 선택 기능
  • ✔ '통역하기' 버튼: API 호출을 실행하는 메인 버튼
  • ✔ Gemini API 연동: 선택된 모드(A, B)에 맞춰 적절한 프롬프트를 전송하고 응답을 받는 로직

1일이라는 한정된 시간 내에 현실적인 완성을 위해 과감히 포기하는 기능입니다.

  • ✖ 지역 공감 (사투리 번역): Mode C, D
  • ✖ 모든 음성 기능 (STT / TTS)
  • ✖ 복잡한 UI/에러 처리: 세밀한 에러 핸들링이나 복잡한 UI 애니메이션은 우선순위에서 제외합니다.

기술 스택

1일 해커톤에 맞춰 가장 빠르고 효율적인 조합으로 구성된 기술 스택입니다.

핵심 성공 전략: 프롬프트

프로젝트의 성패를 좌우할 2가지 핵심 Gemini API 프롬프트 예시입니다. 이 프롬프트를 기반으로 테스트하고 발전시켜야 합니다.

Mode A: 손주어 → 할머니어 (정서 번역) 

"당신은 신조어와 줄임말을 전혀 모르는 할머니에게 손주의 메시지를 번역해주는 '다정한 AI 비서'입니다. 손주의 텍스트를 받으면, 신조어의 뜻을 괄호 안에 풀어서 설명하고, 전체 문장을 할머니가 이해하기 쉽고 정겨운 말투로 재구성해야 합니다. 예를 들어 '킹받네'는 '(왕짜증 났어)'처럼 풀고, '폼 미쳤다'는 '(정말 멋지다)'처럼 풀어주세요. 절대 손주의 말투 그대로 번역하면 안 됩니다. 오직 번역된 문장만 답해주세요."

Mode B: 할머니어 → 손주어 (유머 번역) 

"당신은 할머니의 다정한 메시지를 요즘 MZ세대 손주가 재밌게 받아들일 수 있도록 '유머러스한 신조어'로 번역하는 AI 비서입니다. 할머니의 문맥과 사랑은 유지하되, '많관부', '얼공', '밥무?' 등 최신 유행하는 밈과 신조어를 적극 사용해서 문장을 재구성해야 합니다. 오직 번역된 문장만 답해주세요."
프롬프트가 복사되었습니다!
